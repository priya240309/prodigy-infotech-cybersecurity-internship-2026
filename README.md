# Prodigy Infotech Cybersecurity Internship 2026

## Task 01 – Caesar Cipher

### Description
This project is part of the Cybersecurity Internship at Prodigy Infotech.  
It demonstrates the implementation of the Caesar Cipher encryption and decryption technique using Python.

The Caesar Cipher is one of the simplest and earliest encryption methods. It works by shifting each letter in the plaintext by a fixed number of positions in the alphabet.

### Features
- Encrypt a message
- Decrypt an encrypted message
- User-defined shift value
- Simple command-line interaction

### Technologies Used
- Python

### How It Works
The program asks the user to enter a message and a shift value.  
Each character in the message is shifted according to the given value to produce encrypted text.  
The same shift value can be used to decrypt the message.

### Example
Plain Text: HELLO  
Shift: 3  

Encrypted Text: KHOOR
